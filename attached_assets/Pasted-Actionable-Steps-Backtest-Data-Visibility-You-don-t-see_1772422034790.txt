Add "Close All Positions" Button
Add this button to your trader tab UI (likely in the control panel near "Start Autonomous Trader" and "Test Signal"):

HTML<button id="close-all-positions" style="background: #ff4444; color: white; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer;">
  Close All Positions
</button>
JavaScript (add to your main UI script):
JavaScriptdocument.getElementById('close-all-positions').addEventListener('click', async () => {
  if (!confirm('Close ALL open positions? This is irreversible.')) return;

  try {
    const res = await fetch('/api/trader/close-all', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ account: selectedAccount })
    });
    const data = await res.json();
    alert(data.message || 'Close all command sent');
  } catch (err) {
    alert('Error closing positions: ' + err.message);
  }
});
Backend endpoint (add to routes.ts):
JavaScriptapp.post('/api/trader/close-all', async (req, res) => {
  const { account } = req.body;
  try {
    const payload = `key=${process.env.CROSS_TRADE_KEY};command=closeall;account=${account || 'sim101'}`;
    await sendToCrossTrade(payload);
    res.json({ message: 'Close all command sent to NinjaTrader' });
  } catch (err) {
    res.status(500).json({ error: err.message });
  }
});

Add Daily Loss Limit Control
Add this input to the trader tab UI (near Risk Per Trade):

HTML<label>Daily Loss Limit ($):</label>
<input type="number" id="daily-loss-limit" value="300" min="0" step="50">
JavaScript (add to UI script):
JavaScriptlet dailyLossLimit = 300; // default $300

document.getElementById('daily-loss-limit').addEventListener('change', e => {
  dailyLossLimit = parseFloat(e.target.value);
  console.log(`Daily loss limit set to $${dailyLossLimit}`);
});

// In signal check before emission
if (currentDailyPnl <= -dailyLossLimit) {
  console.log(`Daily loss limit reached: $${currentDailyPnl} / -$${dailyLossLimit}`);
  return;
}
