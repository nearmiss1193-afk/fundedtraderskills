Enable Multi-Signal Convergence on Same Bar (Manual Alignment)
To capture the manual's "multiple concept convergence" (multiple patterns/factors aligning = higher edge), add this convergence check + bonus.Paste into trader.ts (signal emission or post-detection block):

JavaScript// After detecting all patterns for current bar
const barSignals = detectedSignals.filter(s => s.timestamp === currentBar.timestamp && s.symbol === currentSymbol);

if (barSignals.length >= 2) {
  const bonus = (barSignals.length - 1) * 1.5; // +1.5pt per extra pattern
  barSignals.forEach(signal => {
    signal.confluence += bonus;
    signal.notes = `Convergence detected: ${barSignals.map(s => s.pattern).join(', ')} (+${bonus.toFixed(1)}pt)`;
    console.log(`[CONVERGENCE] ${signal.symbol} @ ${signal.timestamp}: ${signal.notes}`);
  });
}
This boosts any signal when ≥2 patterns fire on the same bar (e.g., Wedge + 3bar or Buy Setup + Retest). Test during next RTH.
