Add VWAP (Volume Weighted Average Price) as Dynamic S/R (Manual Ch.3 Volume Extension – Helpful Addition)
Manual emphasizes volume as confirmation for S/R. VWAP is a volume-weighted MA — adds dynamic level for mean reversion (price above VWAP = bullish bias, below = bearish).

Paste this into indicators.ts or addIndicators():
JavaScript// VWAP Indicator (extends manual volume analysis)
function calculateVWAP(bars) {
  let cumPV = 0;
  let cumVolume = 0;

  bars.forEach(bar => {
    const typicalPrice = (bar.high + bar.low + bar.close) / 3;
    cumPV += typicalPrice * bar.volume;
    cumVolume += bar.volume;
  });

  return cumPV / cumVolume;
}

// In confluence calculation
const vwap = calculateVWAP(currentBars);
const aboveVWAP = lastBar.close > vwap;
if (aboveVWAP && signal.direction === 'long') signal.confluence += 1;
if (!aboveVWAP && signal.direction === 'short') signal.confluence += 1;
signal.notes += ` [VWAP ${aboveVWAP ? 'Above' : 'Below'} +1]`;
