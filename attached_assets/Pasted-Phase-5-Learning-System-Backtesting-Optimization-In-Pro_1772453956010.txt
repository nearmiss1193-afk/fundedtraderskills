Order Flow Analysis – Full Integration
Order Flow is now fully incorporated into the system. Since Polygon provides aggregated bars, we approximate true order flow using delta, cumulative delta, absorption, and divergence.
Key Order Flow Rules Added (aligned with Live Traders methodology & standard VP usage):

Bid/Ask Imbalance Approximation: Calculate delta (volume on up bars - volume on down bars). +1pt for positive delta on long signals (buying pressure), +1pt for negative delta on short signals (selling pressure).
Cumulative Delta (CD): Rolling sum of delta over 10–20 bars. +1pt if CD diverges from price (e.g., price new low but CD higher → bullish divergence).
Footprint Proxy: Bar volume split into buy/sell volume based on close/open (close > open = buy volume). +1pt for absorption (high volume but little price move).
Delta Divergence: +2pt bonus if delta diverges from price trend (hidden buying/selling).
Filter: Reject signals if delta is neutral (low imbalance) without volume surge.

JavaScript// Order Flow Confluence - add to calculateConfluence or as separate function
function getOrderFlowConfluence(bars) {
  if (bars.length < 20) return 0;

  const recent = bars.slice(-20);
  let cumulativeDelta = 0;
  let buyVolume = 0;
  let sellVolume = 0;

  recent.forEach(bar => {
    const barDelta = (bar.close > bar.open) ? bar.volume : -bar.volume;
    cumulativeDelta += barDelta;
    if (bar.close > bar.open) buyVolume += bar.volume;
    else sellVolume += bar.volume;
  });

  const avgDelta = cumulativeDelta / recent.length;
  const imbalance = Math.abs(buyVolume - sellVolume) / (buyVolume + sellVolume);

  let ofBonus = 0;
  if (imbalance > 0.6) ofBonus += 1; // strong imbalance
  if (Math.sign(avgDelta) !== Math.sign(recent[recent.length - 1].close - recent[0].open)) ofBonus += 1; // delta divergence
  if (recent[recent.length - 1].volume > averageVolume(recent.slice(-10)) && Math.abs(recent[recent.length - 1].close - recent[recent.length - 1].open) < averageRange(recent.slice(-10)) * 0.5) ofBonus += 1; // absorption

  return ofBonus;
}
